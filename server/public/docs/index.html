Introduction

This is the Documentation to the Favorit-Kock API.

Hier ist die Dokumentation für Favorit-Kock-API.

Base URL: http://localhost

This documentation aims to provide all the information you need to work with the Favorit-Kock API.

Diese Dokumentation enthält alle Informationen, die Sie für die Arbeit mit der Favorit-Kock-API benötigen.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Beim Scrollen sehen Sie im dunklen Bereich rechts Codebeispiele für die Arbeit mit der API in verschiedenen Programmiersprachen (oder als Teil des Inhalts auf Mobilgeräten). Sie können die verwendete Sprache mit den Registerkarten oben rechts (oder über das Navigationsmenü oben links auf Mobilgeräten) ändern.

Authenticating requests

This API is not authenticated.

Admin endpoints

GET Categories

requires authentication

Returns paginated list of categories.

Example request:
curl --request GET \
    --get "http://localhost/api/v1/super-admin/categories?page=1" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/v1/super-admin/categories"
);

const params = {
    "page": "1",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Example response (200):


{"data":{"id":"01h3hkhxrh15atksjr11hrck0d","title":"First category","description":"This is the first category"}, ...}

POST Category

requires authentication

Creates a new Category record.

Example request:
curl --request POST \
    "http://localhost/api/v1/super-admin/categories" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"title\": \"geip\",
    \"description\": \"Voluptas quis porro quaerat unde dolores consequatur.\"
}"

const url = new URL(
    "http://localhost/api/v1/super-admin/categories"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "title": "geip",
    "description": "Voluptas quis porro quaerat unde dolores consequatur."
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
